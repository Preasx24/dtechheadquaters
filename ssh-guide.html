SSH WebSocket Configuration Guide

Learn how to configure SSH WebSocket accounts for secure and stable connections.

What is SSH WebSocket?

SSH WebSocket combines Secure Shell (SSH) protocol with WebSocket technology to enable secure tunneling and bypass network restrictions. It supports SSL/TLS payloads and allows stable internet connections.

SSH WebSocket Configuration Details

SSH WebSocket supports multiple protocols:

  • SSL/TLS: Ideal for encrypted and secure data transmission. Best used on port 443.
  • HTTP: Functions with specific payloads and supports ports like 443 and 8080.
  • TCP: Used for raw connections with custom payloads. Recommended for experienced users.

Note: SSL/TLS is usually the most reliable protocol for SSH WebSocket connections.

How to Create an SSH WebSocket Account on SSHMax

Follow these steps to create an SSH WebSocket account on SSHMax:

  1. Go to SSHMax.
  2. Click on the three lines (Menu) at the top-right corner.
  3. Select SSH WebSocket.
  4. You'll also see an option for SSH Tunnel – feel free to experiment with it.
  5. Choose a server. It's best to select one close to your location, but experimentation can yield better results.
  6. Enter your Username, Password, and Host.
  7. Enable the option to Reserve Host as Address.
  8. Complete the reCAPTCHA verification.
  9. Click on Create Account.
  10. Copy the account details provided. If there’s an SSL/TLS payload, make sure to copy it as well.

Tip: Always use the SSL/TLS payload for better connection stability and security.
