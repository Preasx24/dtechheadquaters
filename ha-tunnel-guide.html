HA Tunnel Plus - Connection and Host Testing Guide

Learn how to use HA Tunnel Plus to test various hosts and secure your internet connection. Follow the steps for SSL/TLS, HTTP, and TCP connections.

SSL/TLS Method

SSL/TLS is a secure method for connecting to hosts by encrypting your traffic and using a certificate-based protocol. This method is useful when you're looking to bypass firewalls and secure connections.

To use this method, enter the host address and enable the SSL/TLS option. After that, you can activate either:

  • Realme Host for an automatic connection to a reliable host.
  • TCP to connect to a TCP server using port 443.

Once configured, test the connection to ensure everything is working correctly. You can choose different servers based on your location for optimal performance.

HTTP Method

HTTP works by routing your connection through an HTTP proxy. This method is straightforward and requires minimal setup. To use this, simply enter the host address with port 443 and enable the HTTP method.

This method is particularly effective when you only need to bypass restrictions with an encrypted connection but do not need a complex configuration.

Video Tutorial: Watch the tutorial on how to use the HTTP method

TCP Method

TCP is a reliable and secure method for connecting to a host via the Transmission Control Protocol. This method involves more detailed configuration and may require selecting a suitable payload for the best connection.

For the best results, experiment with different payloads to see which one works best for your connection. You can enable the TCP option after entering the host and port 443, and test the connection with different server options.
