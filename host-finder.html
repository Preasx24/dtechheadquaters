What Are Zero-Rated Hosts?

Zero-rated hosts are websites and services that offer free access to users without consuming any data from their mobile data plans. These hosts are typically provided by mobile network operators such as MTN, Vodacom, Telkom, and Cell C. By leveraging zero-rated hosts, you can access certain content or services without worrying about data usage.

Our Host Finder tool helps you discover these zero-rated hosts and test them to make sure they are providing the intended free access. We also show how to retrieve headers, test the hosts for SSH, WebSocket (Port 443), and V2Ray, as well as methods for creating accounts with these services.

Here are some simple steps on how you can test and create accounts with these hosts:

  • Test host headers using command-line tools like curl or wget.
  • Use WebSocket testing for more advanced connections that require secure tunneling (Port 443).
  • Create V2Ray and SSH accounts for advanced tunneling and free internet access.
  • Explore alternative methods like using VPNs to route traffic via zero-rated hosts.
