V2Ray Details Guide

Understand V2Ray protocols, learn to create accounts, and configure connections properly.

What is V2Ray?

V2Ray is a versatile proxy tool that enables secure and private internet access. It supports protocols like VLess and VMess, which are ideal for bypassing restrictions and improving connection stability.

VLess Protocol

VLess is a lightweight protocol designed for better performance, lower latency, and efficient data transfer. It works well with SSL/TLS encryption.

VMess Protocol

VMess is a more robust protocol that focuses on authentication and strong encryption. It is widely used for secure proxy connections in restricted networks.

How to Create a V2Ray Account on SSHMax

Follow these steps to create a V2Ray account on SSHMax:

  1. Go to SSHMax.
  2. Click on the three lines (Menu) at the top-right corner.
  3. Select either V2Ray VLess or V2Ray VMess based on your preference.
  4. Choose a server. It’s recommended to pick a server close to your location, but feel free to experiment with other servers.
  5. Enter your desired Username and Host details.
  6. Select the option to Reserve SNI Host as the address.
  7. Complete the reCAPTCHA verification.
  8. Click on Create Account.
  9. Once completed, you'll receive the WebSocket Configuration.
  10. If there are two WebSocket configurations (port 80 and port 443), always choose the one with WebSocket 443 for better security and stability.

Note: Save your WebSocket configuration securely, as it will be required for setting up your V2Ray connection.
